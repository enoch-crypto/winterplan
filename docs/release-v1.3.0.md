# WinterPlan v1.3.0 发版记录

## 版本定位

`v1.3.0` 是日常作息打卡改版。

## 产品方向

从寒假专项计划调整为日常可持续使用的作息打卡工具。

## 主要改动

### 日常化

- 去掉首页、设置和默认模板里的寒假/凛冬/作战文案。
- 应用标题改为 `作息打卡`。
- 默认日程改成日常作息模板。
- `作业` 导航改为 `事项`，默认内容改成健康、学习/工作、生活、兴趣。

### 可编辑作息

- 保留每日时间段编辑。
- 保留项目名称、内容、图标编辑。
- 支持新增、删除和排序日程。
- 每个日程仍可进入计时并完成打卡。

### 月历特殊安排

- 新增 `月历` 页面。
- 展示从今天开始的未来 30 天。
- 点击某一天可查看当天特殊安排。
- 支持给某一天添加特殊事件。
- 支持删除某一天的特殊事件。
- 有事件的日期会显示标记点。

### 数据能力

- 新增 `CalendarEvent` 模型。
- 特殊事件保存到本地。
- 特殊事件纳入撤销、恢复、导出备份、导入备份和重置。

## 验证

- `dart format` 已执行。
- `flutter analyze` 通过。
- `flutter test` 通过。
