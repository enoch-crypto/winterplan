# WinterPlan v1.2.0 发版记录

## 版本定位

`v1.2.0` 是待办清单功能版本。

## 新增功能

### To do list

- 新增底部导航 `待办`。
- 支持输入一条待办并立即添加。
- 支持勾选完成。
- 支持删除单条待办。
- 支持清除已完成待办。
- 显示待完成和已完成数量。

## 数据能力

- 新增 `TodoItem` 数据模型。
- 使用本地存储保存待办。
- 待办数据纳入撤销、恢复、导出备份、导入备份和重置。

- `flutter analyze` 通过。
- `flutter test` 通过。
